# triple_2022_frontend

- 트리플 프론트엔드 과제 전형입니다.

## 실행 방법

- npm i 로 모듈을 설치합니다.
- npm run start로 프로젝트를 실행합니다.

<br/>🍉 실행 후 http://localhost:3000/ 에 접속하면, 이 프로젝트를 확인할 수 있습니다.

## 😊 사용한 라이브러리

### typescript

- 타입스크립트를 사용하여, 타입 에러를 코드 작성 단계에서 미리 확인할 수 있습니다.

### styled-components

- CSS-in-JS 라이브러리입니다.
- CSS를 컴포넌트기반으로 작성할 수 있어 효율적이고 재사용에 좋습니다.
